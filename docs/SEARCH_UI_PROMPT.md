## 五、搜索帮助弹窗内容

### 完整帮助文档

```
📖 知识库搜索帮助

🎯 如何使用

1. 输入关键词
   在搜索框中输入您要查找的内容关键词

2. 查看结果
   系统会显示最相关的文档，按相关度排序

3. 打开文档
   点击文档标题或卡片查看详细内容

💡 搜索技巧

• 使用具体关键词
  ✅ 好的："差旅费报销流程"
  ❌ 不好："那个怎么报销"

• 使用核心词汇
  ✅ 好的："请假"、"考勤"
  ❌ 不好："关于员工请假的有关规定"

• 尝试同义词
  如果没找到结果，尝试换个说法

• 使用筛选功能
  通过部门、类型、时间等条件精确定位

🔍 支持的搜索内容

• 制度规范：公司各项规章制度
• 业务流程：各项业务操作流程
• 技术文档：系统操作指南、技术手册
• FAQ：常见问题解答
• 通知公告：公司重要通知

❓ 常见问题

Q: 搜索不到想要的文档怎么办？
A: 尝试更换关键词、使用筛选功能，或联系知识库管理员

Q: 如何搜索特定部门的文档？
A: 使用"高级筛选"功能，选择对应部门

Q: 搜索结果太多怎么办？
A: 使用筛选功能缩小范围，或使用更精确的关键词

如有其他问题，请联系IT支持：xxx@company.com
```

## 九、代码实现示例

### React 组件示例

```typescript
// SearchPage.tsx 搜索提示词配置

const SEARCH_PLACEHOLDERS = [
  "搜索文档、制度、流程...",
  "试试搜索：报销流程、请假制度、技术文档...",
  "在知识库中查找相关内容..."
]

const SEARCH_TIPS = `
📚 搜索技巧

• 使用准确的关键词效果更好
• 可以搜索文档标题、内容、标签
• 支持搜索制度、流程、技术文档等

💡 试试搜索：
• "报销" - 查找报销相关文档
• "请假" - 查找请假制度和流程
• "系统" - 查找系统使用指南
`

const EMPTY_SEARCH_HELP = (query: string) => `
😕 未找到与"${query}"相关的文档

💡 搜索建议：
• 尝试更换关键词
• 使用更简短的关键词
• 搜索文档的核心内容

🔗 热门搜索：
报销 | 请假 | 考勤 | 系统使用 | 规章制度
`

const HOT_SEARCHES = [
  "报销流程",
  "请假制度",
  "考勤管理",
  "系统使用手册",
  "组织架构",
  "财务制度",
  "人事政策",
  "IT支持",
  "办公指南",
  "安全规范"
]
```

### 搜索框组件

```typescript
<Search
  placeholder={SEARCH_PLACEHOLDERS[0]}
  size="large"
  enterButton={<SearchOutlined />}
  onSearch={handleSearch}
  maxLength={200}
  suffix={
    <Tooltip title={SEARCH_TIPS}>
      <QuestionCircleOutlined style={{ color: '#999' }} />
    </Tooltip>
  }
/>
```

### 空结果提示组件

```typescript
<Empty
  description={
    <div>
      <Paragraph>{EMPTY_SEARCH_HELP(query)}</Paragraph>
      <Divider />
      <Space wrap>
        {HOT_SEARCHES.map(keyword => (
          <Tag
            key={keyword}
            style={{ cursor: 'pointer' }}
            onClick={() => handleSearch(keyword)}
          >
            {keyword}
          </Tag>
        ))}
      </Space>
    </div>
  }
  image={Empty.PRESENTED_IMAGE_SIMPLE}
/>
```
